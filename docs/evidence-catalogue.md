# Evidence Catalogue — Phase 3

> Every decision in this repository must be backed by evidence.
> This catalogue defines what evidence exists, what it contains, and what facts it is authoritative for.

---

## E-001 — AIS (Annual Information Statement)

| Property     | Value |
|--------------|-------|
| ID           | E-001 |
| Issuer       | Income Tax Department |
| Format       | Portal (downloadable as PDF / JSON) |
| Frequency    | Updated throughout the year; final snapshot at year-end |
| Reliability  | High — pulled from third-party reporter data |

### Fields

| Field                  | Section in AIS       | Authoritative For |
|------------------------|----------------------|-------------------|
| Salary Paid            | Part B — SFT-001     | Salary Income |
| Rent Received          | Part B — SFT-002     | House Property Income |
| Dividend Paid          | Part B — SFT-011     | Dividend Income |
| Interest Paid (Bank)   | Part B — SFT-016     | Interest Income |
| Interest Paid (Other)  | Part B — SFT-017     | Interest Income |
| Capital Gains — MF     | Part B — SFT-018     | LTCG / STCG on MF |
| Capital Gains — Equity | Part B — SFT-019     | LTCG / STCG on equity |
| Foreign Remittance     | Part B — SFT-013     | Foreign Income flag |
| TDS — Salary           | Part A               | TDS credit |
| TDS — Other            | Part A               | TDS credit |
| TCS                    | Part A               | TCS credit |

### Notes
- AIS may contain mismatches with 26AS — cross-check mandatory.
- Taxpayer can raise feedback/dispute on AIS entries.
- Always prefer AIS over manual calculations for third-party reported income.

## E-002 — 26AS (Tax Credit Statement)

| Property     | Value |
|--------------|-------|
| ID           | E-002 |
| Issuer       | Income Tax Department |
| Format       | Portal (downloadable as PDF / Text) |
| Frequency    | Updated in real-time as TDS is deposited |
| Reliability  | High — authoritative for TDS credit claims |

### Fields

| Field              | Part in 26AS  | Authoritative For |
|--------------------|---------------|-------------------|
| TDS on Salary      | Part A        | TDS credit for salary |
| TDS on Property    | Part A1       | TDS credit u/s 194IA |
| TDS on Rent        | Part A2       | TDS credit u/s 194IB |
| TCS                | Part B        | TCS credit |
| Advance Tax        | Part C        | Advance tax paid |
| Self-Assessment Tax| Part C        | SAT paid |
| TDS Defaults       | Part D        | Outstanding demand flag |

### Notes
- 26AS is the legally authoritative source for TDS credit.
- AIS supersedes 26AS for income information; 26AS remains primary for tax credit.
- Mismatch between 26AS TDS and Form 16 TDS is a critical validation flag.

## E-003 — Prefill JSON (IT Portal)

| Property     | Value |
|--------------|-------|
| ID           | E-003 |
| Issuer       | Income Tax Department |
| Format       | JSON (downloadable from IT Portal) |
| Frequency    | Annual — available from June |
| Reliability  | Medium — depends on timely reporting by deductors |

### Fields

| Field                   | Authoritative For |
|-------------------------|-------------------|
| Personal Info           | Name, PAN, DOB validation |
| Bank Account Details    | Refund bank account |
| AIS Summary Values      | Pre-filled income heads |
| Carry-Forward Losses    | Previous year losses |
| Previous Refund/Demand  | Outstanding items |

### Notes
- Prefill JSON accelerates filing but must be verified against Form 16 and AIS.
- Do not rely on prefill for capital gains — always use broker statement.

## E-004 — Form 16

| Property     | Value |
|--------------|-------|
| ID           | E-004 |
| Issuer       | Employer |
| Format       | PDF (Part A digitally signed; Part B employer-generated) |
| Frequency    | Annual — issued by June 15 |
| Reliability  | High for salary income and employer TDS |

### Fields

| Field                     | Part    | Authoritative For |
|---------------------------|---------|-------------------|
| Employer TAN              | Part A  | Deductor verification |
| TDS Deposited (quarterly) | Part A  | TDS credit |
| Salary Paid               | Part A  | Gross salary |
| Gross Salary              | Part B  | Salary income |
| Exempt Allowances         | Part B  | HRA, LTA, other exemptions |
| Standard Deduction        | Part B  | ₹50,000 / ₹75,000 |
| Deductions u/s 16         | Part B  | Professional tax, entertainment |
| Net Salary                | Part B  | Taxable salary |
| Chapter VI-A Deductions   | Part B  | 80C, 80D, etc. |
| Tax on Salary             | Part B  | Tax computed by employer |

### Notes
- Part A is generated by TRACES and digitally signed — authoritative.
- Part B is generated by employer — verify all deduction claims independently.
- Form 16 TDS must match 26AS TDS; mismatch is a critical validation flag (V-001).

## E-007 — Capital Gains Statement (Broker / AMC)

| Property     | Value |
|--------------|-------|
| ID           | E-007 |
| Issuer       | Stockbroker or Mutual Fund AMC |
| Format       | PDF or Excel |
| Frequency    | Annual |
| Reliability  | High for securities capital gains |

### Fields

| Field           | Authoritative For |
|-----------------|-------------------|
| Buy Date        | Holding period (LTCG vs STCG) |
| Sell Date       | Transaction date |
| Buy Price       | Cost of acquisition |
| Sell Price      | Sale consideration |
| STCG Amount     | STCG |
| LTCG Amount     | LTCG |
| STT Paid        | STT-paid flag (for u/s 112A eligibility) |
| Scrip / Fund    | Asset identification |

### Notes
- Consolidated Capital Gains Statement (CAS) from CAMS/KFintech covers all MF transactions.
- Use broker statement for equity; CAS for mutual funds.

## Evidence Priority Matrix

When two evidence sources disagree, use this priority order:

| Fact                | Primary Source | Secondary Source |
|---------------------|----------------|-----------------|
| TDS Credit          | 26AS           | Form 16 / 16A |
| Salary Income       | Form 16 Part A | AIS |
| Interest Income     | AIS            | Bank Statement |
| Dividend Income     | AIS            | Dividend Warrants |
| Capital Gains       | Broker Statement | AIS |
| Property TDS        | 26AS (Part A1) | Form 26QB |
| Deductions          | Form 16 Part B | Self-declaration |
| Advance Tax / SAT   | 26AS (Part C)  | Challan |
